# Custom Command Line Arguments
The following arguments can be passed after the double dash `--` argument to modify the game's behavior.
| Command | Description |
| --- | --- |
| `--demo` | Hides the Fullscreen and Quit buttons on the options screen. |
| `--score-log` | Writes all new leaderboard entries to a `score_log.txt` file located in the [user data directory](https://docs.godotengine.org/en/latest/tutorials/io/data_paths.html#accessing-persistent-user-data-user). Each entry will appear on a new line with the format `name - value - suffix`. |
